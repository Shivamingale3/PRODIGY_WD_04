Document Toolbox

An online platform for editing images and creating or compressing PDF files. Our website is built using HTML, CSS, and JavaScript and all the processing takes place in the browser using the browser's JavaScript engine. This means that no data or metadata is transmitted over the internet, and everything happens on the client-side. Our website does not require any login or registration, and it is completely free to use.
